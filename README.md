# Money Manager 💰

A React application to track income and expenses.

## 🚀 Features
- Add income and expenses
- Real-time balance calculation
- Transaction history
- Clean UI

## 🛠 Tech Stack
- React.js
- JavaScript
- CSS

## ▶️ Run Locally
```bash
npm install
npm start
```
